Question-1: What are the different ways to select an element in the DOM?

we can select DOM several way.such as using getElementById,getElementsByClassName,getElementsByTagName,querySelectorAll,querySelector etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

when we need to insert or get HTML content we use innerHTML.when we need to insert or get text inside HTML element we use inneText.

Question-3: What is event delegation in the DOM?

By event delegation in JavaScript we can attach an event listener to a parent element instead of adding individual event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

when an event occurs on a nested element, it “bubbles up” all the way to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

we use createElement() to create an element.we use appendChild() to add an element.we use removeChild() to remove a child from an element.
